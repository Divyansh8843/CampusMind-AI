# 🎓 CampusMind AI
### The Global Intelligent Academic Ecosystem

> **One Platform. Infinite Possibilities. Global Education Redefined.**

**CampusMind AI** is the world's first fully integrated, AI-powered academic platform, designed to unify the college experience. From intelligent lecture notes to global peer networking and career placement, it serves as the single digital campus for students worldwide.

---

## 💡 Motivation
In a fragmented digital world, students juggle multiple platforms for notes, study partners, and job hunting. **CampusMind AI** was born from the vision to **unify** these disconnected tools into one **"Global Campus"**—ensuring that whether a student is in a dorm room or across the ocean, they have the same world-class access to success.

---

## 🌟 Key Features

### 🧠 1. AI Lecture Weaver (Interactive Learning)
*   **Live Audio-to-Notes**: Instantly records live lectures and "weaves" them into structured **Cornell Notes** using advanced speech-to-text AI.
*   **Active Learning**: Chat with your notes, generate quizzes, and get instant summaries.
*   **Smart Study Plans**: AI generates personalized weekly schedules based on student goals and syllabi.

### 🌍 2. Global Peer Match (Real-Time Collaboration)
*   **Skill-Based Connections**: Matches students globally based on overlapping skills (e.g., Python + React) and current study topics.
*   **Live Virtual Study Rooms**: Features **HD Video Calls** and a shared collaborative **Whiteboard** for solving complex problems together.
*   **Community Q&A**: A gamified, StackOverflow-style hub where students ask, answer, and earn **XP/Levels** for helping others.

### 💼 3. Career Catalyst (Placement Ready)
*   **AI Resume Architect**: Upload your resume and get instant, AI-driven rewrites tailored to specific Job Descriptions (JD) for maximum ATS impact.
*   **Mock Interview Simulator**: Practice with an AI interviewer that provides real-time feedback on confidence, clarity, and technical content.
*   **Opportunity Hunter**: Aggregates Jobs, Internships, and Hackathons in one centralized dashboard.

### 🛡️ 4. Admin Command Center
*   **360° Student Analytics**: Comprehensive dashboard showing student activities, document uploads, and engagement stats.
*   **Document Management**: Robust tools to oversee all institutional resources with real-time storage statistics.
*   **Role-Based Security**: Distinct, optimized interfaces for **Admins** (Management) and **Students** (Learning).

## 📦 Installation & Deployment

To deploy the Global Platform locally, follow these steps:

### Prerequisites
*   Node.js (v18+)
*   Python (v3.9+)
*   MongoDB & Redis (Running locally or cloud)

### 1. Clone & Setup
```bash
git clone https://github.com/your-org/campusmind-ai.git
cd campusmind-ai
```

### 2. Backend (Server)
```bash
cd server
npm install
# Create .env file with DB_URI, JWT_SECRET, AWS_KEYS, etc.
npm start
# Server runs on Port 3000
```

### 3. Frontend (Client)
```bash
cd client
npm install
npm run dev
# Dashboard available at http://localhost:5173
```

### 4. AI Microservice
```bash
cd ai-service
pip install -r requirements.txt
python -m app.main
# AI Engine runs on Port 8000
```
